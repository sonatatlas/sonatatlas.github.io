```rust

format!("{argument}", argument = "test");
format!("{name} {}", 1, name = 2);
format!("{a} {c} {b}", a = "a", b = "b", c = 3);

```

+ nothing = Display
+ ? => Debug
+ x? => Debug with lower-case hexadecimal integers
+ o => Octal
+ x => LowerHex
+ p => Pointer
+ b => Binary
+ e => exp
